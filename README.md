<h1>Login com aplicativos internos</h1>

Este projeto é um aplicativo desenvolvido com React Native utilizando Expo, organizado em uma estrutura baseada em screens e React Navigation para gerenciar as rotas.

O sistema de login funciona usando LocalStorage (AsyncStorage), evitando o uso de SQLite, já que o app se conecta a um servidor web e não precisa de banco de dados local.

<h2>Como executar o projeto</h2>

Instale as dependências (disponíveis no arquivo package.json) e rode o app com os comandos padrão do Expo:
```bash
npx expo install @react-navigation/native
```
<br>

```bash
npx expo install react-native-screens react-native-safe-area-context react-native-gesture-handler react-native-reanimated
```
<br>

```bash
npm install @react-navigation/native-stack
```
<br>

```bash
npx expo start
```
<hr>

Atente-se às dependências do Navigations, Screens e Picker.
